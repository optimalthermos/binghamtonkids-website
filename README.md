# Children's Charity of Greater Binghamton

Modern, responsive website for CCGB — a 501(c)(3) nonprofit serving children in the Greater Binghamton, NY area.

## Features

- Fully responsive design (mobile-first)
- Light/dark mode with system preference detection
- Embedded Instagram feed
- Scroll animations
- FAQ accordion
- Google Maps integration
- SEO-optimized with Open Graph tags
- Accessible (WCAG AA compliant)

## Deploy on Railway

1. Fork or clone this repo
2. Connect to [Railway](https://railway.com)
3. Deploy — uses the included `Dockerfile` (nginx)

The site is purely static HTML/CSS/JS — no build step required.

## Local Development

```bash
# Serve locally (requires npx or any static file server)
npx serve . -l 3000
```

## Content

- **About**: Mission statement, community impact statistics
- **Programs**: Easter Egg Hunt (in memory of Harper Stantz), Give Back Friday, School Supply Drive
- **News**: Latest events, Instagram embed, event recaps
- **Board**: Board of Directors listing
- **FAQ**: Common questions about donating, volunteering, events
- **Contact**: Email, mailing address, social media, Venmo, Google Maps
